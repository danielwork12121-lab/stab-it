================================================================================
 PROJECT PURPOSE
================================================================================
StabIt / 忧忧 lets users pin worries, chat with AI, review later, then remove or reschedule pins.
The app loop must always return to Home/Stuffy after removal or reschedule.

================================================================================

================================================================================
 API CONTRACTS
================================================================================

POST /api/ai/chat (3min timeout)
Request:
{
  "mode": "pinning" | "review",
  "messages": [{"role": "user" | "assistant", "content": "text"}],
  "pin": {coreIssue, reflectionDays, warmExplanation, currentGuides, aiResult, reviewHistory, reviewCount, createdAt, reviewStage, pendingReviewChoice}
}

Response (pinning):
{
  "reply": string,
  "readyToPin": boolean,
  "readyToRemove": false,
  "analysis": {safe, coreIssue, reflectionDays, warmExplanation, currentGuides}
}

Response (review):
{
  "reply": string,
  "readyToPin": false,
  "readyToRemove": boolean,
  "review": {stillAffectsUser, reasonCategory, nextReflectionDays}
}

POST /api/ai/analyze-worry (15s timeout)
Request: {"userText": string}
Response: {safe, coreIssue, reflectionDays, warmExplanation, currentGuides}
